# 项目介绍

## 概述

PrismAvaInMobilePlatform 是一个基于 **Avalonia** 的 Android 跨平台应用模板。

Avalonia 是 .NET 领域最成熟的跨平台 UI 框架，支持 Windows、macOS、Linux、Android、iOS 和 Web。

## 核心依赖

| 包名 | 版本 | 说明 |
|------|------|------|
| Avalonia | 11.3.13 | 跨平台 UI 框架 |
| Semi.Avalonia | 11.3.7.3 | UI 主题 |
| Microsoft.Maui.Essentials | 10.0.51 | 原生设备功能 |

## 支持的平台

- **Android**: 移动设备和平板电脑
