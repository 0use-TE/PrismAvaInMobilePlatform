# PrismAvaInMobilePlatform

基于 **Avalonia** 的 Android 跨平台应用模板。

## 技术栈

- Avalonia 11.3 + Prism.DryIoc 9.0
- Microsoft.Maui.Essentials (原生设备功能)
- Semi.Avalonia 主题
- Serilog 日志

## 已实现功能

- 拍照 / 录像
- 文件选择 (单选/多选)
- 手电筒控制
- 震动反馈
- 浏览器跳转
- 设备信息
- 主题切换
- 窗口通知

## 快速开始

```bash
dotnet run -f net10.0-android
```
